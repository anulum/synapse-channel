# Support

## Getting help

- **Documentation.** Start with the [`README.md`](README.md), the
  [`TEAM_PROTOCOL.md`](TEAM_PROTOCOL.md) wire reference, and
  [`ARCHITECTURE.md`](ARCHITECTURE.md).
- **Bugs and feature requests.** Open an issue using the templates on GitHub.
  Include the version (`synapse --version`), what you expected, and what
  happened.

## Security

Do not report security issues in public. Follow [`SECURITY.md`](SECURITY.md).
